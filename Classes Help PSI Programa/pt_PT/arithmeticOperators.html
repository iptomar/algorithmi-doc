Operadores Aritméticos

Os operadores aritméticos estão presentes diariamente na nossa vida, e podemos dividi-los em três grupos.

Inteiros: São operadores que utilizam exclusivamente variáveis ou valores inteiros.

 Operadores disponíveis para números inteiros:

     +    Adição

     -    Subtracção

     /    Divisão inteira

     *    Multiplicação

     %    Resto da divisão inteira

     ^    Potêncialização

Reais: São operadores que utilizam variáveis ou valores numéricos em que pelo menos um elemento é do tipo real.

 Operadores disponíveis para números reais:

     +    Adição

     -    Subtracção

     /    Divisão inteira

     *    Multiplicação

     ^    Potêncialização

Texto: São operadores que utilizam variáveis ou valores do tipo texto.

 Operadores disponíveis para texto:

     +    Concatenação de texto
