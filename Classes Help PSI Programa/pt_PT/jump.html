Jump

O Jump pode ser interpretado por:

Break

Descrição

A instrução break serve para terminar uma sequência de instruções dentro de um ciclo.

Este é usado em ciclos de repetição while, do/while, for.

Quando o break é usado em ciclos de repetição, este, causa uma interrupção imediata, continuando a execução do programa na próxima linha depois do ciclo.

Continue

Descrição

A instrução continue finaliza o ciclo que está a ser executado e salta diretamente para a próxima execução.

Esta instrução é apenas utilizada em ciclos de repetição.

Quando este é executado, o ciclo volta imediatamente para o local da condição do ciclo de repetição.
